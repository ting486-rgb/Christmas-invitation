# 🎄 圣诞团建活动邀请函

一个充满圣诞特效的团建活动邀请函系统，包含礼物心愿配对功能。

## ✨ 特色功能

- 🎅 **登录特效**：圣诞老人骑鲁道夫飞过，礼物散落雨
- ❄️ **背景特效**：持续雪花飘落和星星闪烁
- 🔔 **互动铃铛**：可点击的圣诞铃铛特效
- 🎁 **心愿配对**：每个人可以看到自己抽到的心愿
- 👨‍💼 **管理员面板**：完整的配对信息查看

## 🚀 在线访问

访问地址：`https://YOUR_USERNAME.github.io/christmas-invitation`

## 👥 使用方法

1. 输入你的英文名登录（如：liztang, tingtingong 等）
2. 查看你抽到的圣诞心愿
3. 管理员使用 `admin` 账户查看所有配对信息
